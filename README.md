# Socks-rest
Тестовое задание.  
Автоматизация учета склада носков. CRUD-приложение на Spring Boot  
## Стек
- Java 17, Spring Boot 3.0
- Spring JPA
- Spring Validation
- Liquibase
- PostgreSQL
- Lombok
- Docker
- TestContainers, Mockito, AssertJ, JUnit5, WebFlux
## Как запустить
- mvn clean package
- docker compose up -d
